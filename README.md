### Project: Fitness MERN App

This is a fitness app that let's you log fitness data

## Technologies Used:

- MongoDB
- Express
- React
- NodeJS

### Getting Started:
```
# Clone the repository
git clone https://github.com/mujtabachoudhry/full-stack-coursework/
# Install dependencies
npm install
# Start the app
npm start
```
